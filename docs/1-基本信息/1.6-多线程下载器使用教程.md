# 多线程下载器使用指南

### 什么是多线程下载器?

多线程下载器通过将文件拆分为多个数据块并行下载，充分利用网络带宽，大幅提升下载速度与稳定性。它适用于大文件、批量资源以及网络波动环境，支持断点续传和并发管理，帮助用户更高效、可靠地完成下载任务。

::: tips 提示

本文仅介绍NDM和Gopeed,其余多线程下载器不做对比介绍

:::

##### 下载器配置

打开NDM下载器,点击 Settings,将线程数改至32(推荐),"OK"保存
