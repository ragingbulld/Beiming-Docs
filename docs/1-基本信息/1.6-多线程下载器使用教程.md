# 多线程下载器使用指南

---

## 什么是多线程下载器？

多线程下载器通过将文件拆分为多个数据块并行下载，充分利用网络带宽，大幅提升下载速度与稳定性。  
它适用于大文件、批量资源以及网络波动环境，支持断点续传和并发管理，帮助用户更高效、可靠地完成下载任务。

::: tip 提示
本文仅介绍 **NDM** 和 **Gopeed**，其余多线程下载器不做对比介绍。
:::

打开 NDM → 点击 **Settings** → 将线程数修改为 **32（推荐）** → 点击 OK 保存

### 创建下载任务

以下载「北冥-1.21.1生电」整合包为例：

1. 点击下载
2. 浏览器弹出下载框
3. 右键任务 → 复制下载链接
4. 在 NDM 中点击 **New URL**
5. 粘贴链接开始下载

::: warning 友情提示

- 初次尝试建议从 **32 线程** 开始测试  
- 速度未达上限时，每次增加 **8–16 线程**  
- 达到带宽上限后不要继续增加线程  
- 线程过高可能导致服务器限速或封禁  

合理配置线程，才能稳定高速下载。

:::
