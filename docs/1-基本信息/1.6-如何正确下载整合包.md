## 什么是多线程下载器？

多线程下载器通过将文件拆分为多个数据块并行下载，充分利用网络带宽，大幅提升下载速度与稳定性。  
适用于大文件、批量资源及网络波动环境，支持断点续传与并发管理。

::: tip 说明
本文仅介绍 **NDM** 与 **Gopeed** 两款下载器，其余产品不作对比。
:::

<a id="线程优化建议"></a>
# ⚙ 线程优化建议

| 网络情况 | 推荐线程 |
| -------- | -------- |
| 较差网络 | 4 - 8    |
| 普通网络 | 8 - 16   |
| 良好网络 | 16 - 32  |
| 高速网络 | 32 - 64  |
| 极速网络 | 64+      |

---

<a id="友情提示"></a>
::: warning 友情提示
- 建议从 **32 线程** 作为起点  
- 若速度未达上限，每次增加 **8–16 线程**  
- 达到带宽极限后不要继续增加线程  
- 线程过高可能触发服务器限速或封禁  

合理设置线程，才能稳定且高速下载。
:::
