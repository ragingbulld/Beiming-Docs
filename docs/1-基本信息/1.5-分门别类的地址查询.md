### 第二步：首选接入方案（智能分流）

**推荐三大运营商玩家优先使用。** 系统会自动检测你的宽带运营商，并分配延迟最低的节点。

- **IPv4 地址：** `v4.mc-x.beiming.games`
- **IPv6 地址：** `v6.mc-x.beiming.games`

> **构造示例：** 如果你想玩 **生电群组服** (编号为 1)，直接输入：`v4.mc-1.beiming.games` 

#### ⚠️ 重要提示：非三大运营商玩家须知

- **无法解析风险：** 如果你是**海外玩家**、或使用**非三大运营商**（如：长城宽带、广电网络、教育网/校园网），智能分流地址可能无法解析。
- **解决办法：** 请直接跳过第二步，前往 **第三步：备用接入方案**，手动尝试连接三条线路中延迟最低的一条。

------

### 第三步：备用接入方案（手动运营商线路）

如果智能地址无法连接、解析失败或波动较大，请根据你自家的**宽带运营商**手动选择对应的固定线路。

#### 📶 手动线路构造汇总

| 运营商类型 | IPv4 构造格式 | IPv6 构造格式 |
| :--- | :--- | :--- |
| **中国电信** | `dx-v4.mc-x.beiming.games` | `dx-v6.mc-x.beiming.games` |
| **中国联通** | `lt-v4.mc-x.beiming.games` | `lt-v6.mc-x.beiming.games` |
| **中国移动** | `yd-v4.mc-x.beiming.games` | `yd-v6.mc-x.beiming.games` |
------

### 🛡️ 线路访问控制策略（防火墙规则）

为了优化网络质量并精准防御，各线路设有严格的 **IP 入站过滤规则**。请务必对号入座，否则会导致连接超时（Timed Out）。

| 接入线路          | 允许接入的客户端 IP                                          | 拦截（丢弃）的 IP  |
| ----------------- | ------------------------------------------------------------ | ------------------ |
| **电信线路 (dx)** | **中国电信**、**海外玩家**、**非三大运营商** (如长城/广电/教育网) | 中国联通、中国移动 |
| **联通线路 (lt)** | **中国联通**、**海外玩家**、**非三大运营商** (如长城/广电/教育网) | 中国电信、中国移动 |
| **移动线路 (yd)** | **中国移动**、**海外玩家**、**非三大运营商** (如长城/广电/教育网) | 中国电信、中国联通 |

#### 🌍 海外及非三大运营商（长城/广电/校园网等）连接指南

由于你们的 IP 不在移动、联通、电信的黑名单拦截范围内，**理论上三条线路都可以连接**。但由于不同骨干网的物理路由不同，延迟表现会有巨大差异：

1. **测试方法：** 请分别尝试连接 `电信`、`联通`、`移动` 三个地址。
2. **选择标准：**
   - 优先选择**进入游戏最快**、**丢包率最低**的那条线路。
   - **通常规律：** 海外玩家建议优先尝试 **电信 (dx)** 或 **联通 (lt)** 线路，通常国际出口更优。
   - **小宽带玩家：** 若你是广电或长城宽带，建议优先尝试与其有合作关系的线路（通常是联通或移动）。

### ❓ 常见问题与网络排查

- **我不确定是否有 IPv6？** 请访问 **[ipw.cn](https://ipw.cn)** 进行快速检测。如果没有显示IPv6 地址，请务必使用带 **`v4`** 的地址连接。
- **我不清楚自己的运营商？** 访问 [ip138.com](https://www.ip138.com) 即可查看当前宽带的所属供应商（电信/联通/移动）。
- **连接频繁掉线？** 部分地区运营商的 DNS 劫持会导致“智能分流”失效。此时请尝试使用第三步中的 **手动线路** 绕过自动检测。
