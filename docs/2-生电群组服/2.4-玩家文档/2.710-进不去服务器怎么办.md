# 进不去服务器怎么办

::: tip 关于跨版本

由于1.21的mojang改变了网络架构，所以尽管服务器有via跨版本插件，1.21及以上的客户端如果mod加多了可能无法连接服务器，还请使用纯净端或本服提供的客户端整合包

:::

如果遇到服务器连接没亮，或者显示红x，请更改本地DNS缓存（大群群文件中有脚本bat）
