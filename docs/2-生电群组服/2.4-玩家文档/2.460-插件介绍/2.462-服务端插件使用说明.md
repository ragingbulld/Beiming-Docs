# MCDR 插件使用教程

## 概览

使用 `!!help` 获取帮助。

## 权限

- `!!MCDR permission set 2` 设置玩家的身份组为 `helper`。
- `!!MCDR plugin reloadall` 插件重载。

## Mirror 插件

- `!!op` 获取 OP 权限。
- 指令前缀默认为 `!!mirror`，用于控制多个镜像服务器。通过指令前缀区分，详情请查看配置文件。

常用指令：

- `!!mirror` 显示指令帮助。
- `!!mirror help` 显示指令帮助。
- `!!mirror status` 获取镜像服务器实例运行状态（未知/已停止/正在停止/正在启动/正在运行）。
- `!!mirror start` 启动镜像服务器实例。
- `!!mirror stop` 停止镜像服务器实例。
- `!!mirror kill` 强制终止镜像服务器实例（需要权限三）。
- `!!mirror sync` 执行文件同步。
- `!!mirror confirm` 确认某指令的操作。
- `!!mirror reload` 热重载镜像服务器配置文件。

## Spectator_tp

- `!!tp <维度> <坐标>`，如 `!!tp 0 0 0 0` 即传送到主世界（Overworld）0,0,0的坐标。
- 也可以不输入维度和坐标。

## Where2go

- `!!here` 向所有人发送自己的坐标，并自己变成太阳。
- `谁谁谁在哪` 发送查询某个玩家的坐标，并让该玩家变成太阳。

## Location list

- `!!loc list` 显示所有位置列表。
- `!!loc add <名称> here` 添加当前位置到列表。
- `!!loc add <名称> <x> <y> <z>` 添加指定坐标到列表。

## World_eater_manage

适用于原版服务器的世吞假人管理插件。

常用指令：

- `!!we` 查看帮助信息。
- `!!we spawn <维度> <服务器视距> <组名>` 在指定维度生成假人组，服务器视距表示假人生成区域范围。
- `!!we rspawn <区块半径> <服务器视距> <组名>` 以玩家所在区块为中心生成假人组。
- `!!we kill <组名>` 删除指定假人组并使其退出游戏。
- `!!we clear` 删除所有假人组。
- `!!we restart <组名>` 使该组假人在服务器重启后自动生成。
- `!!we list` 查看正在运行的假人组。
- `!!we reload` 重载插件。

## PB（Backup）插件

- `!!pb` 查看插件简介。
- `!!pb help` 查看帮助。
- `!!pb list` 显示已备份的列表。
- `!!pb make <备注>` 创建备份。

## CB（Chunk Backup）插件

- `!!cb` 显示帮助信息。
- `!!cb help` 显示全部指令。
- `!!cb make <区块半径> <注释>` 创建玩家所在区块中心的区块备份。
- `!!cb make -s <区块半径> <注释>` 创建静态备份（即永久备份）。
- `!!cb dmake <维度> <注释>` 备份指定维度的所有区块。
- `!!cb dmake -s <维度> <注释>` 创建静态备份的维度备份。
- `!!cb pmake <x1> <z1> <x2> <z2> in <维度> <注释>` 备份指定坐标范围内的区块。
- `!!cb back <槽位>` 回档指定槽位的区块。
- `!!cb back -s <槽位>` 回档静态备份的区块。
- `!!cb restore` 恢复存档到回档前的状态。
- `!!cb del <槽位>` 删除指定槽位的备份。
- `!!cb list` 显示备份槽位信息。
- `!!cb list -s` 查看静态备份槽位信息。
- `!!cb show <槽位>` 显示指定槽位的所有信息。
- `!!cb show -s <槽位>` 显示静态备份的所有信息。
- `!!cb force_reload` 强行重载插件。

### 自定义备份

- `!!cb custom create <自定义备份名>` 创建一个自定义备份对象。
- `!!cb custom list` 列出所有自定义备份对象。
- `!!cb custom save <自定义备份名>` 保存自定义备份对象。
- `!!cb custom save -s <自定义备份名>` 保存为静态备份。
- `!!cb custom show <自定义备份名> page <页数>` 分页查看自定义备份的子槽位列表。
- `!!cb custom del <自定义备份名>` 删除自定义备份对象。

## 回档相关操作

- `!!cb rollback` 查看上次回档的详细信息。
- `!!cb confirm` 再次确认是否回档。
- `!!cb abort` 在任何时候中断回档操作。
