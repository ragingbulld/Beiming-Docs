# 一句话总结（先看）

要实现“回国梯”：**购买支持回国节点的套餐（注意：仅英文名的“国外用户套餐/纯英文名套餐”包含回国节点） → 获取订阅链接/节点列表 → 在客户端（如 V2RayN / Clash / V2RayNG / Shadowrocket）导入订阅 → 选择带“回国”/China 的节点并开启系统代理**。回国节点通常流量计费倍率很高（文档称为 5×），流量消耗注意估算。([help.ednovas.me][1])

# 详细步骤（逐步）

## 1) 选套餐与付款（非常重要）

1. 在官网/导航页查看套餐，**如果你人在国外并需要“回国节点”必须购买“纯英文名的国外用户套餐”**（文档里明确写了要买英文名套餐才有回国节点）。回国节点是高成本节点，文档说明这些节点计为 5 倍流量（例如你用 1GB 实际扣 5GB）。因此请注意流量与计费。([help.ednovas.me][1])
2. 支付方式：支持支付宝、USDT（虚拟币）、信用卡等。若支付出现问题，请通过 Telegram/Discord 联系客服（help 文档给了 TG 群和私聊）。注意：当前文档显示**退款策略有限制**（有时不支持退款），购买前确认。([help.ednovas.me][1])

**小贴士**：若你只需要回国节点短期试用，可先买最小流量包或月付最低档，避免一次买太多被 5× 抽干流量。

---

## 2) 注册 / 获取订阅（下单后）

1. 购买后，服务通常会把**“订阅链接”**（用于客户端自动拉节点）或单独的节点信息（VMess/VLESS/Trojan/Shadowsocks）发到你的注册邮箱或提供在控制面板。help 页面说明订阅链接是客户端导入的关键。([ednovas.me][2])
2. 如果订阅链接不能直接用于你的客户端，文档建议使用 **订阅转换工具**（例如 ACL4SSR / 订阅转换站）进行一次转换或联系 TG 求助。([help.ednovas.me][3])

---

## 3) 在常见设备上安装并配置客户端（按你用的系统选）

> **重要**：不管用哪个客户端，关键点是 **导入订阅（或添加节点） → 选择“回国节点”/中国节点 → 开启系统代理（或相应的路由/分流规则使流量走该节点）**。下面给出常用客户端的具体操作要点与参考文档。

### A. Windows（推荐：V2RayN 或 Clash for Windows）

* 推荐阅读与下载地址见官网帮助。V2RayN 用法要点：

  1. 下载并解压 V2RayN（或 V2RayN + core），运行 v2rayN.exe。([help.ednovas.me][4])
  2. 打开 **订阅设置 → 新增订阅**，粘贴你从控制面板拿到的订阅链接，保存并“更新订阅”以拉取节点。([help.ednovas.me][4])
  3. 在节点列表里找到带“回国”或 China / CN 的节点（help 提到回国节点会单独列出），选中它。
  4. 点击 **系统代理 → 自动配置系统代理 / 开启**（V2RayN 里通常在托盘或主界面右下角），此时系统流量会走所选节点。([help.ednovas.me][4])
* 如果用 **Clash for Windows / Proxifier 配合**：导入 Clash 的订阅（Profiles → 订阅链接 → Download），在 Profiles 里选相应节点组/节点，切换到“全局”或按需要设置分流规则使中国流量走代理（详见 Clash 文档）。([help.ednovas.me][5])

**特别说明（回国流量）**：很多默认分流规则是“绕过中国大陆”，也就是中国 IP 不走代理 —— 这对“回国梯”反而不符合你的需求。要实现回国效果，**确保你选中的节点本身是中国节点，并且客户端/路由规则不要把中国 IP 排除在代理之外**（即不要启用“绕过中国大陆”的规则，或改为全局/仅代理中国 IP 的规则，或直接选择该节点并开启系统代理）。([help.ednovas.me][5])

---

### B. macOS（ClashX）

1. 下载 ClashX（首次会提示未知开发者，允许打开）。建议安装 Helper（Install），以便切换代理不每次输入密码。([help.ednovas.me][6])
2. 状态栏点击 ClashX → 配置 → 托管配置 → 管理 → 添加订阅，粘贴你的订阅链接并下载。([help.ednovas.me][6])
3. 选择订阅/节点组后，选择对应的回国节点并开启代理。注意与 Windows 同样要确认分流规则（不要绕过中国大陆）。([help.ednovas.me][6])

---

### C. Android（V2RayNG / Clash for Android）

* **V2RayNG**（文档推荐）：

  1. 在左侧菜单选择订阅设置 → + 添加订阅 → 粘贴你的 v2ray 订阅链接 → 更新订阅。
  2. 测试节点延迟，选择回国节点并连接。
  3. 在设置 → 路由模式中，如果默认是“绕过局域网及中国大陆”，请改为**代理全部流量或确保中国流量会被代理**，否则回国节点不会被用于访问中国网站。([help.ednovas.me][7])

* **Clash for Android**：导入订阅（Profiles → 粘贴订阅 → 下载），选择节点组后在主界面点击“开始代理”。Clash 可设置“分流”规则，按需调整以确保回国流量走代理。([help.ednovas.me][8])

**提示**：Android 上有些应用（如微信、银行客户端）可能检测代理/VPN，连接后如有异常可尝试切换不同回国节点或临时关闭代理后再试。

---

### D. iOS（Shadowrocket / Quantumult / Kitsunebi）

* iOS 上常用的是 **Shadowrocket**（付费 App）或 Quantumult / Kitsunebi。使用方法为：添加配置/订阅 → 下载节点 → 选择回国节点并开启代理。help 文档有相应教程（Shadowrocket）。（iOS 客户端各自界面不同但思路相同）([help.ednovas.me][6])

---

### E. 路由器 / 软路由（回国软路由设置）

* 如果你想把家里路由器整个流量都走回国节点（适合局域网内所有设备都表现为中国 IP），help 中有“回国软路由设置教程”。步骤通常是：在软路由（OpenWrt / Padavan / LEDE / BIRD）上安装 Clash/Xray 等服务，导入订阅并设置路由/分流，把目标设备流量导到回国节点。因为具体固件与界面繁多，建议按 help 的专页教程操作或联系 TG 求助。([help.ednovas.me][4])

## 4) 常见问题与排查（快速参考）

* **订阅导入失败**：用订阅转换工具（ACL4SSR 等）转换后再导入；或把订阅发给客服让他们帮你转换。([help.ednovas.me][3])
* **访问国内网站仍然是国外 IP / 无法回国**：检查是否选择了回国节点（节点名称/说明通常会标注回国），并确认客户端的分流规则没有“绕过中国大陆”；如果客户端默认绕过，请调整为“全局代理”或专门创建规则把中国 IP 走代理。([help.ednovas.me][1])
* **流量消耗快**：回国节点文档标注为 5 倍计费（即耗流量快），注意套餐流量与倍率；如需要长期回国建议买更大流量包。([help.ednovas.me][1])
* **节点不稳定 / 不能登录付款后遇问题**：联系 TG 群或私聊客服，或在 TG/Discord 群里反馈（help 页给了联系方式），他们会在文档例举的回复时间内处理。([help.ednovas.me][1])

## 5) 合规与使用注意事项（必须看）

* help 文档反复强调 **禁止滥用与分享订阅**（比如公开发布到群组会被封号），并列出了滥用类型（BT、邪教网站等）会被直接封号，购买与使用请遵守服务条款和当地法律。([help.ednovas.me][1])
