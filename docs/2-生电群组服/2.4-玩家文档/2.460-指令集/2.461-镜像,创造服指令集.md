# 镜像/创造服指令集

!!op 获取op

指令前缀默认为!!mirror, 控制多个镜像服时将通过指令前缀区分, 详见配置文件

!!mirror 显示指令帮助

!!mirror help 显示指令帮助

!!mirror status 获取镜像服实例运行状态，状态未知/已停止/正在停止/正在启动/正在运行

!!mirror start 启动镜像服实例

!!mirror stop 停止镜像服实例

!!mirror kill 强制终止镜像服实例（需要权限三）

!!mirror sync 进行文件同步

!!mirror confirm 确认某指令的操作

!!mirror reload 热重载对应镜像服的配置文件
