# 假人管理手册

### 1. 假人使用规范

所有人只允许有和自己同名的一个私有假人，除了常在线假人的不管是快递还是储存物品，都只允许存在三分钟，滞留站的假人如不是在较远距离（如史莱姆切门）需要来回往返不得常驻。

### 2. 常用假人举例

- **food**：放置食物类物品
- **tools**：放置工具类物品  
- **weapons**：放置武器类物品

### 3. 假人挂机规范

机器挂机需用专有假人挂机，不得使用随意命名的假人，统一使用以下命名规范：gj，gj_1，gj_2

等序列化假人名称

如不满足条件则使用自己私有假人/小号或向管理申请，如发现私自召唤假人则按违反服规处理。

### 4. 违规处理

如果因私自使用新名称假人影响其他玩家游玩，按违反服规处置。

::: tip 注

**常在线假人定义**：如全物品、伪和平、破基岩机、世吞等的加载假人

**特殊假人规范**：世吞、破基岩加载假人用 `gj_1` 至 `gj_n`，不得单独使用如 `st`、`bedrock` 等名称假人（使用 `!!we` 召唤，详情参考[插件使用教程](/2-生电群组服/2.4-玩家文档/2.460-插件介绍/2.461-服务端插件使用说明.md)）
