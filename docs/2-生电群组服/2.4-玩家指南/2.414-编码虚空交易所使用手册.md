# 编码虚空交易是使用手册

## 使用指南

::: tip 温馨提示

如果交易需求不是很大，可以直接跳进**茅坑主体**面对面和对应奸商交易

以下内容为完整使用教程

:::

##### 翻牌子

在**箱子UI单选器**右侧的箱子中选择想要进行交易的村民

![image-20251212173941305](https://bu.dusays.com/2025/12/12/693be2dccca31.webp)

以下内容以牧师（经验瓶）为例

![image-20251212174059993](https://bu.dusays.com/2025/12/12/693be32ceb235.webp)

取走一个经验瓶，放入音符盒右侧的投掷器内，点击音符盒

![image-20251212174117285](https://bu.dusays.com/2025/12/12/693be33d4f053.webp)

![image-20251212174133566](https://bu.dusays.com/2025/12/12/693be34eb3e9c.webp)

##### 开始交易

村民会进坑，你也进坑，然后打开面前的活板门让水流冲走你

在激流勇进的途中，右键村民打开交易栏，进入折跃门后进行交易

> 别直接交易熬

此时交易不会消耗该村民交易的次数，可以长期交易

如果想利用autotrade进行挂机交易，请跳转另外一则mod使用教程-[autotrade使用指南](https://sodamc.com/47309-46061111.html)
