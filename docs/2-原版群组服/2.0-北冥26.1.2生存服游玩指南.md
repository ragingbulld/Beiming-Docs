# 北冥 26.1.2 生存服游玩指南

适用服务器：北冥 26.1.2 混合登录生存服  
服务端版本：Lophine 26.1.2  
游玩模式：困难生存，正版和离线玩家都能进入  
默认端口：25588  
QQ群：1041762935

这是一份给玩家看的入服和游玩手册。重点是怎么进服、怎么注册登录、怎么传送、圈地、交易和使用常用功能。

## 进服前准备

推荐使用 26.1.2 客户端进入。服务器允许正版玩家和离线玩家一起玩，服务器地址以群公告或管理员发布为准。如果地址没有写端口，连接失败时把端口补成 `25588`。

首次进入后，系统会要求注册账号。注册成功前不能正常聊天、移动和使用大部分命令，这是账号保护，不是卡服。

## 注册与登录

首次进服输入：

```text
/register 密码 密码
```

再次进服输入：

```text
/login 密码
```

密码长度为 6 到 32 位，只建议使用英文、数字和常见符号。不要用 `123456`、`password` 这种弱密码。同一个 IP 默认最多注册 3 个账号。正版玩家也需要密码登录，当前不开放 `/premium` 免密模式。

已经登录后想改密码，使用：

```text
/changepassword 旧密码 新密码
```

离线玩家请固定使用同一个名字进服。大小写乱改、名字换来换去，都会被当成不同账号或触发登录保护。

## 新人第一天

进服后先完成登录，再输入：

```text
/menu
```

菜单里有常用入口。建议先看服务器指南、地图、费用说明和领地功能。开荒前可以用 `/rtp` 随机传送到野外，找好位置后先圈地，再放箱子和机器。

主世界、下界和末地都可以圈地。不要把家建在别人领地边缘，也不要贴着主城造高频机器。

## 传送、家和返回

本服使用轻量传送插件。普通玩家可以使用 TPA、家、随机传送、主城和返回功能。

| 功能 | 命令 | 说明 |
|---|---|---|
| 请求传送到玩家 | `/tpa 玩家名` | 每次请求消耗 20 金币 |
| 请求玩家传送到你身边 | `/tphere 玩家名` | 每次请求消耗 20 金币 |
| 接受传送 | `/tpaccept` | 收到请求后使用 |
| 拒绝传送 | `/tpdeny` | 不想接受时使用 |
| 随机传送 | `/rtp` | 下界禁用，主世界开荒常用 |
| 设置家 | `/sethome 家名` | 默认最多 3 个家 |
| 回家 | `/home 家名` | 不写家名时通常回默认家 |
| 查看家列表 | `/homes` | 查看已设置的家 |
| 删除家 | `/delhome 家名` | 清理不用的家 |
| 回主城 | `/spawn` | 回到服务器主城 |
| 返回上一个位置 | `/back` | 用于死亡或传送后返回 |

传送有等待时间。普通传送等待 3 秒，命令冷却 30 秒。传送前不要乱动，避免取消。

## 领地系统

领地用来保护建筑、箱子、红石机器和农场。先圈地，再开始长期建设。

| 功能 | 命令 | 说明 |
|---|---|---|
| 圈地 | `/claim` | 打开圈地流程或界面 |
| 查看领地 | `/claims` | 查看自己的领地 |
| 放弃领地 | `/unclaim` | 删除当前或指定领地 |

普通玩家默认最多 1 个领地，总计最多 25 个区块。首次圈地需要 500 金币，扩展每个区块需要 120 金币。普通玩家单个领地最多 25 个区块，最多 5 名成员。

默认保护会阻止外人破坏、开箱、拿取掉落物、伤害实体和乱动重要方块。进入或离开领地时会有提示。别人领地里不能动的东西，不要用漏洞或机器去绕。

## 金币、官方商店和拍卖

经济系统由 EssentialsC 提供。金币用于传送、领地、玩家商店和玩家交易。

| 功能 | 命令 | 说明 |
|---|---|---|
| 查看余额 | `/balance` | 查看自己的金币 |
| 转账 | `/pay 玩家名 金额` | 给别人付款 |
| 财富榜 | `/baltop` | 查看余额排行 |
| 官方商店 | `/shop` | 购买常用物资 |
| 回收物品 | `/sell` | 出售可回收物资 |
| 查看价格 | `/worth` | 查看手上物品价值 |
| 拍卖场 | `/ah` 或 `/auction` | 玩家之间挂牌交易 |
| 垃圾桶 | `/trash` | 丢弃不用物品 |

官方商店分为建材、红石生电、农牧食材、矿物资源、刷怪笼、工具装备、战斗补给、附魔书和杂项补给。缺基础物资时先看 `/shop`，想卖材料时先试 `/sell` 和 `/worth`。

## 玩家箱子商店

玩家商店使用 QuickShop-Hikari。玩家可以用箱子、陷阱箱、末影箱和木桶创建商店。创建商店需要 10 金币，改价需要 20 金币。交易会有基础税率，用来压住无成本倒卖。

常见做法是拿着要出售或收购的物品点击箱子，按聊天提示输入价格和数量。商店创建后，其他玩家可以通过箱子或牌子购买。商店库存不足、价格写错或余额不足时，按游戏内提示处理。

建议把玩家商店建在自己领地内。公共市场可以集中建设，但箱子一定先确认权限和补货方式。

## 背包便利和工具台

LeafInventory 提供背包便利功能。潜影盒、末影箱、工作台、切石机、砂轮、制图台、织布机、附魔台、熔炉、高炉和烟熏炉等功能已启用。普通玩家不需要额外权限。

潜影盒和常用工具台优先用背包内交互。具体点击方式以游戏内提示为准。大型存储仍建议放在领地内，别把贵重物资只放在随身潜影盒里。

## 地图、死亡点和睡觉

输入 `/bmmap` 查看网页地图入口。地图默认 Web 端口是 `8080`，如果外网地址没有开放，以群公告为准。

死亡后输入 `/deathpoint` 可以查看最近死亡位置。输入 `/coords` 可以查看当前坐标，方便组队集合和记录基地位置。

服务器启用 50% 睡觉规则。在线玩家里有一半睡觉，就能跳过夜晚。遇到夜晚刷怪压力大时，可以在聊天里喊人睡觉。

## 生电、建筑和原理图

服务器保留生存、生电和建筑向玩法。PCA、Servux 和 SyncmaticaP 做了服务端桥接，适合需要共享原理图、材料列表和协作备货的玩家。相关入口可以尝试：

```text
/bms
/syncmaticap
```

这类功能需要客户端配合，具体以群内客户端包和管理员公告为准。本服当前不开放假人玩法，也不要依赖 Carpet 假人规则来设计机器。

大型机器建议远离主城和密集住宅区。高频红石、刷怪塔、村民交易所、区块加载和超大实体农场要控制规模。机器有异常卡顿时，先主动停机，再找管理员处理。

## 游玩规矩

不要盗窃、破坏、恶意圈地、恶意堵门、恶意刷屏、恶意卡服。不要绕过领地权限开箱、推方块、炸机器或偷实体。公共设施用完复原，公共农场按规则补种。

玩家交易自己谈清价格和交付方式。贵重交易建议截图留证。拍卖场和箱子商店标价写错，一般按实际交易结果处理，管理员不保证每次都能回档。

发现复制、刷物、崩服、权限绕过等问题，别公开传播，直接找管理员。主动上报通常按贡献处理，利用漏洞获利会清收益或封禁。

## 常见问题

| 问题 | 处理 |
|---|---|
| 进服后不能动 | 先 `/login 密码` 或 `/register 密码 密码` |
| 忘记密码 | 在群里找管理员，不要反复乱试 |
| TPA 提示金币不足 | `/tpa` 和 `/tphere` 每次 20 金币 |
| 不能破坏方块 | 可能在别人领地里，换地方或找领地主人 |
| 箱子打不开 | 可能被领地保护或商店保护 |
| `/rtp` 不能用 | 下界禁用随机传送，换主世界使用 |
| 地图打不开 | 先用 `/bmmap`，外网地址以群公告为准 |
| 玩家列表中文乱码 | 这是控制台编码显示问题，不影响游戏 |

## 管理员常用提醒

玩家进服问题优先看登录状态。玩家不能动、不能聊天、背包看不到，大多是没登录。玩家建家问题优先让他 `/claim`，不要裸放箱子。玩家说卡顿，先问位置、维度、机器类型和实体数量，再决定是否需要现场处理。

本指南按当前 MCSM 包配置编写。后续如果加入跨版本、反作弊、皮肤站、群组服或更多插件，指南需要同步更新。
